# 更新履歴

## v1.3.0

- マイページ (`/mypage all`) の表で難易度・順位が正しい順番でソートされるようになりました
- 曲名をクリックすると LR2IR のランキングページに移動できるようになりました

## v1.2.0

- マイページ (`/mypage all`) がファイル添付ではなく URL で閲覧できるようになりました（リンクは24時間有効です）
- `/changelog` コマンドで Bot の更新情報を確認できるようになりました

![Discordスクショ](https://raw.githubusercontent.com/tam1252/IR_bot/main/img/demo0.png)
![IRページ](https://raw.githubusercontent.com/tam1252/IR_bot/main/img/demo1.png)

## v1.1.0

- `/announce` でイベントを告知すると、スコアタチャンネルが「開催中のスコアタ」カテゴリーに自動で追加されるようになりました
- 新しいイベントが告知されると、前回のチャンネルが回数に応じたアーカイブカテゴリー（例: `1-10`、`11-20`）に自動で移動するようになりました
- アーカイブ先のカテゴリーが存在しない場合、自動で作成されるようになりました
- 「一般」チャンネルへの @everyone 告知が自動で投稿されるようになりました
- チャンネル名に難易度の ★ が含まれるようになりました（例: `79_★4_thats_the_style!!`）

## v1.0.0

- `/register [LR2ID]` — Discord アカウントと LR2ID を紐づけて登録できます
- `/mypage [回数]` — 指定した回の自分の順位・スコア・BPI を確認できます
- `/mypage all` — 全回の成績を一覧で確認できます
- `/bpi [曲名] [スコア]` — BPI を計算できます
- `/result [回数]` — 指定回のランキングを表示します（管理者用）
- `/announce` — イベント情報を告知します（管理者用）
